1. Blockchain APIs (Balance & Transactions)

These APIs are used to fetch wallet balances and broadcast transactions on mainnet. All are FREE with no API keys required.

Bitcoin - Blockstream API FREE MAINNET

Used in: src/lib/blockchain.ts, src/lib/transactions.ts, server/transactions.js

Base URL: https://blockstream.info/api

Public Bitcoin blockchain API for address info, UTXOs, and transaction broadcasting.

EndpointMethodPurpose
/address/{address}GETGet address balance and stats
/address/{address}/utxoGETGet unspent transaction outputs
/tx/{txid}/hexGETGet raw transaction hex
/txPOSTBroadcast signed transaction

Code Example:

// Get BTC balance const response = await fetch(`https://blockstream.info/api/address/${address}`); const data = await response.json(); const balance = (data.chain_stats.funded_txo_sum - data.chain_stats.spent_txo_sum) / 100000000; // Get UTXOs for spending const utxos = await fetch(`https://blockstream.info/api/address/${address}/utxo`); // Broadcast transaction await fetch('https://blockstream.info/api/tx', { method: 'POST', body: signedTxHex });

Rate Limits: No strict limits, but be reasonable. For production, consider running your own node.

Documentation: https://github.com/Blockstream/esplora/blob/master/API.md

Ethereum - LlamaRPC FREE MAINNET

Used in: src/lib/blockchain.ts, src/lib/transactions.ts

RPC URL: https://eth.llamarpc.com

Free Ethereum JSON-RPC endpoint. Used with ethers.js for balance queries and transaction sending.

MethodPurpose
eth_getBalanceGet ETH balance
eth_callCall contract (e.g., balanceOf for ERC20)
eth_sendRawTransactionBroadcast signed transaction
eth_gasPriceGet current gas price
eth_estimateGasEstimate gas for transaction

Code Example:

import { ethers } from "ethers"; // Create provider const provider = new ethers.JsonRpcProvider("https://eth.llamarpc.com"); // Get ETH balance const balance = await provider.getBalance(address); console.log(ethers.formatEther(balance)); // Send ETH transaction const wallet = new ethers.Wallet(privateKey, provider); const tx = await wallet.sendTransaction({ to: toAddress, value: ethers.parseEther("0.1") }); // Get ERC20 balance (USDT) const contract = new ethers.Contract(USDT_ADDRESS, ERC20_ABI, provider); const usdtBalance = await contract.balanceOf(address);

Rate Limits: Generous free tier. For high volume, use Alchemy or Infura (paid).

Alternatives (Paid):

  • Alchemy: https://alchemy.com (300M compute units/month free)
  • Infura: https://infura.io (100K requests/day free)
  • QuickNode: https://quicknode.com

Solana - Mainnet RPC FREE MAINNET

Used in: src/lib/blockchain.ts, src/lib/transactions.ts

RPC URL: https://api.mainnet-beta.solana.com

Official Solana mainnet RPC. Used with @solana/web3.js for balance and transactions.

MethodPurpose
getBalanceGet SOL balance in lamports
sendTransactionSend signed transaction
getSignatureStatusCheck transaction status

Code Example:

import { Connection, PublicKey, LAMPORTS_PER_SOL } from "@solana/web3.js"; const connection = new Connection("https://api.mainnet-beta.solana.com"); // Get SOL balance const balance = await connection.getBalance(new PublicKey(address)); console.log(balance / LAMPORTS_PER_SOL); // Send SOL const signature = await sendAndConfirmTransaction(connection, transaction, [keypair]);

Rate Limits: Rate limited. For production, use:

  • Helius: https://helius.xyz (free tier available)
  • QuickNode Solana: https://quicknode.com

Tron - TronGrid API FREE MAINNET

Used in: src/lib/blockchain.ts, server/transactions.js

Base URL: https://api.trongrid.io

Official Tron blockchain API for TRC20 tokens and smart contract interactions.

EndpointMethodPurpose
/v1/accounts/{address}/tokensGETGet TRC20 token balances
/wallet/triggersmartcontractPOSTCreate smart contract transaction
/wallet/broadcasttransactionPOSTBroadcast signed transaction

Code Example:

// Get TRC20 USDT balance const USDT_CONTRACT = "TR7NHqjeKQxGTCi8q8ZY4pL8otSzgjLj6t"; const response = await fetch( `https://api.trongrid.io/v1/accounts/${address}/tokens?token_id=${USDT_CONTRACT}` ); // Create TRC20 transfer const tx = await fetch("https://api.trongrid.io/wallet/triggersmartcontract", { method: "POST", headers: { "Content-Type": "application/json" }, body: JSON.stringify({ owner_address: fromAddress, contract_address: USDT_CONTRACT, function_selector: "transfer(address,uint256)", parameter: encodedParams, fee_limit: 100000000 }) });

Rate Limits: 1,000 requests/day free. For more, get API key from https://www.trongrid.io/

2. Market Data APIs

CoinGecko API FREE

Used in: src/lib/marketData.ts, src/lib/transactions.ts

Base URL: https://api.coingecko.com/api/v3

Free cryptocurrency market data API for live prices, market caps, and trending coins.

EndpointPurpose
/simple/priceGet current price for specific coins
/coins/marketsGet market data with 24h changes
/search/trendingGet trending coins

Code Example:

// Get ETH price in USD const response = await fetch( "https://api.coingecko.com/api/v3/simple/price?ids=ethereum&vs_currencies=usd" ); const data = await response.json(); console.log(data.ethereum.usd); // e.g., 3000 // Get market data for multiple coins const markets = await fetch( "https://api.coingecko.com/api/v3/coins/markets?vs_currency=usd&ids=bitcoin,ethereum,solana" );

Rate Limits: 10-50 requests/minute (varies). For more, use Pro API (paid).

Documentation: https://www.coingecko.com/api/documentation

3. Payment Gateway APIs

Important: API Keys Required

Payment gateway APIs require authentication and are typically PAID services with per-transaction fees. Store credentials securely in environment variables.

Interswitch Payment Gateway PAID API KEY REQUIRED

Used in: server/interswitch.js, src/lib/interswitch.ts

Base URL: https://qa.interswitchng.com (Sandbox)
Production: https://saturn.interswitchng.com

Nigerian payment gateway for card payments (Visa, Mastercard, Verve). Requires merchant account.

EndpointMethodPurpose
/passport/oauth/tokenPOSTGenerate access token
/api/v3/purchasesPOSTInitiate card purchase
/api/v3/purchases/otps/authsPOSTVerify OTP
/api/v3/purchases/otps/resendPOSTResend OTP
/api/v3/purchasesGETCheck transaction status

Environment Variables Required:

INTERSWITCH_CLIENT_ID=your_client_id INTERSWITCH_CLIENT_SECRET=your_client_secret INTERSWITCH_CARD_CLIENT_ID=your_card_client_id INTERSWITCH_CARD_SECRET=your_card_secret

Code Example:

// Generate access token const credentials = Buffer.from(`${CLIENT_ID}:${CLIENT_SECRET}`).toString('base64'); const tokenResponse = await fetch('https://qa.interswitchng.com/passport/oauth/token', { method: 'POST', headers: { 'Authorization': `Basic ${credentials}`, 'Content-Type': 'application/x-www-form-urlencoded' }, body: 'grant_type=client_credentials' }); // Initiate card purchase const purchaseResponse = await fetch('https://qa.interswitchng.com/api/v3/purchases', { method: 'POST', headers: { 'Authorization': `Bearer ${accessToken}`, 'Content-Type': 'application/json' }, body: JSON.stringify({ customerId: "customer@email.com", amount: 10000, // in kobo (NGN 100) transactionRef: "unique_ref_123", currency: "NGN", authData: encryptedCardData }) });

Test Cards:

  • Visa: 4000000000002503 (Exp: 01/25, CVV: 111)
  • Mastercard: 5123450000000008 (Exp: 01/39, CVV: 100)

Pricing: Per-transaction fees (contact Interswitch for rates)

Documentation: https://developer.interswitchgroup.com/
